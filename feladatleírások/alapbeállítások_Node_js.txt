
Node.js-hez az alap csomagok telepítése egyszerűen elvégezhető az npm (Node Package Manager) segítségével. Az alábbi lépések segítenek abban, hogy egy alap Node.js projektet összeállíts:

1. Inicializáld a projektet
Nyiss egy terminált abban a mappában, ahol a projekted lesz, majd futtasd:

bash
Kód másolása
npm init -y
Ez létrehoz egy package.json fájlt alapértelmezett beállításokkal.

2. Telepítsd az alapvető csomagokat
a) Express.js (webszerver keretrendszer)
Futtasd:

bash
Kód másolása
npm install express
b) Dotenv (környezet változók kezelése)
Ha a jelszavakat, adatbázis beállításokat .env fájlban szeretnéd tárolni:

bash
Kód másolása
npm install dotenv
c) MySQL (ha adatbázist használsz)
Ha MySQL-t fogsz használni az adatbázishoz:

bash
Kód másolása
npm install mysql
d) Body-Parser (adatok fogadása POST kérésekből) – már az Express része, de külön is telepítheted:
bash
Kód másolása
npm install body-parser
e) Nodemon (fejlesztési eszköz)
Nodemon automatikusan újraindítja a szervert, ha módosítod a kódot:

bash
Kód másolása
npm install --save-dev nodemon

6. További ajánlott csomagok
Cors: Ha más domainekről érkező kéréseket kell kezelni.
bash
Kód másolása
npm install cors
Mongoose: Ha MongoDB-t használsz MySQL helyett.
bash
Kód másolása
npm install mongoose
EJS: Ha sablonmotorra van szükséged a nézetekhez.
bash
Kód másolása
npm install ejs
